Browser Storage Demo

Web browsers can store information locally and retain the stored information even after the web page is closed or the browser is shut down. This page demonstrates the Web Storage API, a simple tool for storing a web page's information in the browser.

Put Items in Storage

Type some text into the input box and click "Store Item" to put it in local storage. Do this several times, and then click on the View Items in Storage button to see the stored values.

See What's in Storage

Once you have some items in storage, go ahead and close this browser and reopen it. Come back to this page and click on the View Items in Storage button. You will see that the browser kept the data even after it was closed.

Clear Storage

click on the Empty Storage button to empty the browser's storage.

Why this is Important

We don't want our scouting application to lose data if the tablet loses its network connection. We'll want the scouting application to save all data entered by the user and send the data once the network connection is restored. With the Web Storage API, we can save scouting data in the browser until we receive a successful response from the server.

How this Works

See the detailed comments in the source code for templates/child-templates/storage.html and static/js/storage.js to see how this works.
