Overview

This is a demonstration of several technologies that will be used in the Issaquah Robotics Society's scouting system for 2022.

This webpage is a simple Flask application. Instructions for running this application and detailed code explanations are available in the source code files. Start by reading the comments in templates/index.html.

There are several navigation tabs across the top of this page, with each tab linking to a different section. Each section describes a different technology that will be used in the scouting system.

The source HTML for this tab is in the templates/overview.html file.
